1st Parallel Version of Pagerank Algorithm. 

If the executable is not already there you can build it by typing:

make

To run type 

bsub<script.bash

The results are shown on results.txt
Type cat results.txt 

If you are compiling this on discovery cluster of Northeastern University then you have to load the following modules:

 1) gnu-4.4-compilers     2) fftw-3.3.3            3) platform-mpi          4) oracle_java_1.7u40    5) gnu-4.8.1-compilers   6) openmpi-1.8.3         7) cuda-6.5              8) matlab_dce_2013b
